# Kebijakan Privasi - SipSip Water Tracker

**Terakhir diperbarui:** 30 Desember 2024

## Pendahuluan

SipSip Water Tracker ("kami", "aplikasi ini") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda saat menggunakan aplikasi mobile kami.

Dengan menggunakan aplikasi SipSip Water Tracker, Anda menyetujui pengumpulan dan penggunaan informasi sesuai dengan kebijakan ini.

---

## Informasi yang Kami Kumpulkan

### 1. Informasi Akun Pengguna
Kami mengumpulkan informasi berikut saat Anda mendaftar atau menggunakan aplikasi:
- **Nama lengkap**
- **Alamat email**
- **Foto profil** (opsional)
- **Informasi kesehatan**: berat badan, usia, jenis kelamin, kondisi kesehatan
- **Preferensi**: target hidrasi harian, waktu bangun, waktu tidur

### 2. Data Penggunaan Aplikasi
- **Catatan konsumsi air**: jumlah air yang diminum, waktu minum
- **Statistik hidrasi**: persentase target harian, streak (hari berturut-turut)
- **Pengaturan pengingat**: waktu dan frekuensi notifikasi
- **Preferensi aplikasi**: bahasa, tema, pengaturan suara

### 3. Informasi Perangkat
- **Identifikasi perangkat**: Device ID untuk notifikasi push
- **Sistem operasi**: versi Android/iOS
- **Token notifikasi**: untuk mengirimkan pengingat minum air

---

## Bagaimana Kami Menggunakan Informasi Anda

Kami menggunakan informasi yang dikumpulkan untuk:

1. **Menyediakan dan meningkatkan layanan**
   - Menghitung target hidrasi yang dipersonalisasi
   - Melacak progres konsumsi air harian Anda
   - Menyediakan statistik dan analisis mingguan/bulanan

2. **Mengirimkan notifikasi**
   - Pengingat untuk minum air sesuai jadwal yang Anda tetapkan
   - Notifikasi pencapaian target harian

3. **Autentikasi dan keamanan akun**
   - Memverifikasi identitas Anda saat login
   - Menjaga keamanan akun Anda

4. **Sinkronisasi data**
   - Menyimpan data Anda secara aman di cloud
   - Memungkinkan akses dari berbagai perangkat

## Layanan Pihak Ketiga

Aplikasi ini menggunakan layanan pihak ketiga yang dapat mengumpulkan informasi untuk mengidentifikasi Anda:

### 1. **Google Play Services**
- **Digunakan untuk:** Infrastruktur dasar aplikasi Android dan autentikasi Google Sign-In
- **Informasi yang dikumpulkan:** Device ID, alamat email Google, nama profil
- **Kebijakan Privasi:** [https://policies.google.com/privacy](https://policies.google.com/privacy)

### 2. **Firebase Authentication**
- **Digunakan untuk:** Manajemen akun pengguna dan autentikasi (termasuk Google Sign-In)
- **Informasi yang dikumpulkan:** Email, nama, User ID, metode autentikasi
- **Kebijakan Privasi:** [https://firebase.google.com/support/privacy](https://firebase.google.com/support/privacy)

### 3. **Firebase Cloud Messaging (FCM)**
- **Digunakan untuk:** Mengirimkan notifikasi push (pengingat minum air)
- **Informasi yang dikumpulkan:** Token perangkat, data notifikasi
- **Kebijakan Privasi:** [https://firebase.google.com/support/privacy](https://firebase.google.com/support/privacy)

### 4. **Google Analytics for Firebase**
- **Digunakan untuk:** Memahami penggunaan aplikasi dan meningkatkan kualitas layanan
- **Informasi yang dikumpulkan:** Data penggunaan aplikasi (anonim), interaksi pengguna
- **Kebijakan Privasi:** [https://firebase.google.com/support/privacy](https://firebase.google.com/support/privacy)

## Penyimpanan dan Keamanan Data

### Penyimpanan Data
- **Data lokal**: Disimpan di perangkat Anda menggunakan Hive (database lokal) untuk akses offline
- **Data cloud**: Disimpan di server backend kami dengan enkripsi untuk sinkronisasi

### Keamanan
Kami menggunakan langkah-langkah keamanan yang wajar untuk melindungi data Anda, termasuk:
- Enkripsi data saat transit (HTTPS)
- Autentikasi berbasis token
- Akses terbatas ke server backend

**Namun**, tidak ada metode transmisi melalui internet atau penyimpanan elektronik yang 100% aman. Kami tidak dapat menjamin keamanan absolut.

## Hak Anda

Anda memiliki hak untuk:

1. **Mengakses data pribadi Anda**
   - Anda dapat melihat semua data Anda di dalam aplikasi (profil, statistik, riwayat)

2. **Mengubah atau memperbarui data**
   - Anda dapat mengedit informasi profil kapan saja melalui menu Profil

3. **Menghapus akun Anda**
   - Anda dapat menghapus akun Anda melalui menu Pengaturan → Hapus Akun
   - Setelah penghapusan, semua data Anda akan dihapus secara permanen dari server kami dalam 30 hari

4. **Menolak notifikasi**
   - Anda dapat menonaktifkan notifikasi melalui pengaturan aplikasi atau pengaturan sistem perangkat Anda
